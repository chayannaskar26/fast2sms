# Fast2sms
Unofficial code for sending sms using Fast2sms Api

# How To

1. Create an account in Fast2Sms.
2. Log In and Go to the API tab
3. Under Api Tab select token
4. Copy Token and run code
5. Paste the token
6. Enter number of receiver
7. Enter message
